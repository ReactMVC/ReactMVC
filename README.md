# ReactMVC
The PHP framework for backend developers

# About ReactMVC


# Documention
Download ReactMVC: 
```
git clone https://github.com/ReactMVC/ReactMVC.git
```
Go to folder:
```
cd ReactMVC
```
set your php version 8.0 or higher.

Download Libraries:
```
composer install
```
Be sure to run it on the domain or subdomain or localhost + port.
